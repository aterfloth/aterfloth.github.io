instructions to make time patterns.

Above you can find the time that this page was loaded at.
See it as being broken up into:

ab : cd : ef : ghi

Rules:

if {a+b= even number or 0} then shapes used in pattern will be curved/circular.
if {a+b= odd number} then shapes used in pattern will be angled/straight.
Use {cd * 10} as width in pixels.
Use {ef * 10} as height in pixels.
Use {g} as background color.
Use {h} as primary color.
Use {i} as secondary color.

See below for colors.
